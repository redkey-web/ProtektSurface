Hero Section Effects
Option 1: Tinted Overlay on Photo

Full-width hero image (bright Sydney cityscape or home)
Dark tinted overlay (30-50% opacity) mimicking actual window tint
Creates dramatic contrast with white text on top
Shows the product effect literally

Option 2: Split Screen Before/After

Left side: Bright/harsh sunlight through window
Right side: Same scene with tint applied (cooler tones)
Vertical divider you can drag to compare
Interactive and shows value instantly

Option 3: Gradient Tint Effect

Start with clear/bright at top
Gradually darken down the page (like tint darkening)
Subtle but reinforces the product visually

Texture Overlays
For Window Tint:
css.hero {
  background: url('hero-image.jpg');
  position: relative;
}

.hero::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: linear-gradient(135deg, 
    rgba(20, 20, 40, 0.7), 
    rgba(10, 10, 30, 0.5));
  mix-blend-mode: multiply; /* Creates tinted glass effect */
}
For Marble Protection:

Subtle marble texture as background (10-20% opacity)
Use white/light gray marble for clean, premium feel
Layer behind content, not distracting

Modern Design Patterns
Glass Morphism Effect:
css.card {

  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.2);
  /* Mimics frosted/tinted glass */
}
